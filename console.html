This page exists for Harry's Light-weight Linux Developer Distro

This distro includes
  • Python
  • php
  • Git
  • Ruby
  • Atom
  • Chrome, Firefox, Opera, Firefox and IE
  • Apache Open Office
  • GIMP (photo editor)
  • Slack and Asana desktop modules

The project goal is to create a bootable OS between 5 and 10mb (the typical lightweight OS is around 100mb)
